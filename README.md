## Description

Framework based on Directed Acyclic Graphs (DAGs) for the reproducible and robust generation of a citation network of every scientific publication of the motor learning literature until the year 2025. It relies on pixi for environment management, Apache Hamilton for the DAGs and DVC for data version control.

 ## Installation 

 To install the environment:
 ```bash
 pixi install
 ```
